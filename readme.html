Load Media From Production Plugin for WordPress

A super simple way to load your media files from another server. No need to copy or download gigabytes of media. Use on your local development or to quickly fire up staging area without duplicating thousands of media file.

How to use

Works with Lando, Local by Flywheel, Docker, Vagrant, MAMP and many more local development environments

.

How to use

define ( 'LOAD_MEDIA_FROM_PRODUCTION_URL', 'https://production-url.com' ); wp cli commands wp load-media-from-production get wp load-media-from-production set https://production-url.com wp load-media-from-production reset Just use a free plugin? If you want a free solution, check out "BE Media from Production" from Bill Erickson. However this does not work with all site installs (like Lando), and doesn't work with CSS images and some URLs. Instead it tries to parse image strings in the content and in attachment functions. This means that custom image display functions - like a responsive ACF image function or pulling image files from the theme - won't work. https://wordpress.org/plugins/be-media-from-production/ Just use htaccess? You could just use a rewrite rule in your .htaccess file, but some local development environments ignore those files (running on nginx). You also have the issue that you could accidentally commit that htaccess file and mess up your live or development site. So this is not an elegant solution. https://gist.github.com/thetwopct/7fef629cf0206cf9642be9c42b28730a

Settings

There are no global settings for the block, each block can be customised as you need.

Available settings are:

  • Alignment - content width, wide width, full width (Theme setting dependent)
  • Video Title - add the title for accessibility and SEO purposes
  • Video Start Time - set a YouTube video start time in seconds
  • Controls - hide the controls for the video once it is playing
  • Loop - loop the video after it has finished playing
  • Thumbnail - turn off webP thumbnail (Older YouTube videos may not have a webP thumbnail)

Instructions for Install

Download the plugin zip file.

Go to WordPress Admin > Plugins > Add New Plugin > Upload Plugin, and upload the plugin zip file.

There are other ways to install the plugin, please see the WordPress documentation.

Once installed, Activate the plugin.

Go to Settings > Load Media From Production to set the URL of your production WordPress install.

Requirements

The plugin is for WordPress, so requires WordPress to be installed.

The plugin will work with sites using Classic Editor, Gutenberg Block Editor, Elementor and more.
